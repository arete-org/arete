Invite Daneel

The public invite link is being finalised with Discord. Until then, you can still experiment with Daneel by running your own instance and connecting it to a development server.

  1. Clone the repository and follow the environment instructions in the README.
  2. Create a Discord application with bot permissions and copy its client ID.
  3. Run the docker compose stack or the Node services directly to bring Daneel online.
  4. Authorise your bot using the OAuth URL builder with the scopes documented in the README.

Once the production invite is ready, this page will redirect you automatically. For now, it keeps curious visitors from hitting a broken link.
